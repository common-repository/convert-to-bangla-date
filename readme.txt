This is simple Plugin for Convert date English to Bangla.

== Description ==

This plugin will convert all english date to bangla. You can use this plugin very easily, just active plugin. Enjoy

== Installation ==

1. Upload `Convert to Bangla Date` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently asked questions ==

= Can i change English date by using this plugin =

Yes, Just Active this plugin and than start working.
